### Q1. What does the regular expression `r"^[A-Z]"` match?
<sub>正则表达式 `r"^[A-Z]"` 匹配什么？</sub>

- A) A string that starts with an uppercase letter <sub>以大写字母开头的字符串</sub>
- B) A string that ends with an uppercase letter <sub>以大写字母结尾的字符串</sub>
- C) Any uppercase letter at the beginning of the string <sub>字符串开头的任意大写字母</sub>
- D) Any character that is not an uppercase letter <sub>任何非大写字母的字符</sub>

**答案：AC**

**考点：** `^` 锚点 + 字符集 `[]`

- `^` 在字符集外表示**字符串开头**
- `[A-Z]` 表示任意大写字母
- 组合起来：匹配以大写字母开头的字符串

> **选项辨析：**
> - A "以大写字母开头的字符串" — 描述的是匹配**条件**：只有以大写字母开头的字符串才能被匹配到
> - C "字符串开头的任意大写字母" — 描述的是实际**匹配到的内容**：只是那个大写字母本身（如 `re.search(r"^[A-Z]", "Hello")` 返回的是 `"H"` 而非 `"Hello"`）
> - 两个选项从不同角度描述了同一个正则，都正确

### Q9. Which method only returns the first match of the pattern in the string?
<sub>哪个方法只返回字符串中模式的第一个匹配？</sub>

- A) `re.match()` <sub>从开头匹配第一个</sub>
- B) `re.findall()` <sub>返回所有匹配列表</sub>
- C) `re.search()` <sub>搜索第一个匹配</sub>
- D) `re.replace()` <sub>（不存在此函数）</sub>

**答案：A**

**考点：** `re.match()` vs `re.search()` vs `re.findall()`

- `re.match()` — 从字符串**开头**匹配，返回第一个匹配（或 None）
- `re.findall()` — 返回所有匹配的列表
- `re.search()` — 搜索整个字符串，返回第一个匹配
- 注意：`re.match()` 只在开头匹配，`re.search()` 在任意位置找第一个

> **此题答案给的是 A**，但严格来说 `re.search()` 也只返回第一个匹配。区别在于 `re.match()` 限定在字符串开头。

### Q11. What is the result of `re.split(r"\W+", "Words, words, words.")`?
<sub>`re.split(r"\W+", "Words, words, words.")` 的结果是什么？</sub>

- A) `['Words', ' ', 'words', ' ', 'words', '.']` <sub>保留空格和标点</sub>

**答案：D**

- `\W+` 匹配一个或多个非单词字符（逗号、空格、句号）
- 字符串被逗号+空格分割，最后的句号后面是空字符串 `''`
- 结果：`['Words', 'words', 'words', '']`

### Q12. Which special sequence matches the start of a string?
<sub>哪个特殊序列匹配字符串的开头？</sub>

- A) `$` <sub>字符串结尾</sub>
- B) `\A` <sub>字符串绝对开头</sub>

- `$` 匹配结尾
- `\Z` 匹配字符串结尾

> **注意：** `^` 也能匹配字符串开头，但 `\A` 是更严格的"仅匹配字符串绝对开头"的特殊序列。

### Q13. Which of the following would match a string that starts with 'a' and ends with 'z'?
<sub>以下哪个匹配以 'a' 开头以 'z' 结尾的字符串？</sub>

- A) `r"^a*z$"` <sub>以 0 个或多个 a 开头，以 z 结尾</sub>
- B) `r"^a+z$"` <sub>以 1 个或多个 a 开头，以 z 结尾</sub>
- C) `r"^a.*z$"` <sub>以 a 开头，中间任意字符，以 z 结尾</sub>
- D) `r"a.*z"` <sub>a 后跟任意字符再跟 z（无锚点约束）</sub>

**答案：C**

**考点：** `^`、`$`、`.*` 组合

- `^a` — 以 a 开头
- `.*` — 中间任意字符（0 或多个）
- `z$` — 以 z 结尾
- A 选项 `^a*z$` 中 `a*` 表示 0 个或多个 a，不能匹配中间有其他字符的情况
- D 选项缺少 `^` 和 `$`，不要求字符串以 a 开头或以 z 结尾

## C. CODE DEBUGGING（代码调试）

### 题目

You are provided with a Python function designed to count the repetitions of a specific word in a text file. However, the code contains 5 errors that affect its functionality. Your task is to identify and correct these errors to ensure the function works as intended.

<sub>以下是一个 Python 函数，用于统计文本文件中特定单词的重复次数。但代码包含 5 个错误，找出并修正它们。</sub>

```python
import re

def count_repetitions(filename, 'word '):
    try:
        with open(filename, encoding='ascii') as f:
            content = f.read()
    except FileExistsError:
        print(f"The file {filename} does not exist.")
    except UnicodeDecodeError:
        print(f"Encoding issue in {filename}")
    else:
        words = re.split(r"[ \w'-]+", content.lower())
        num_repetitions = words.count(word.lower())
        print(f"In the file {filename}, the word appears {word}")

count_repetitions('alice.txt', 'Wonderland')
```

---

### 5 个错误及修正

#### 错误 1：参数名不能是字符串字面量（第 2 行）

```python
# 错误
def count_repetitions(filename, 'word '):

# 修正
def count_repetitions(filename, word):
```

**解释：** 函数参数必须是合法的标识符（变量名），不能是字符串字面量 `'word '`。原代码中参数名带引号和空格，完全无效。

---

#### 错误 2：异常类型错误（第 6 行）

```python
# 错误
except FileExistsError:

# 修正
except FileNotFoundError:
```

**解释：** 文件不存在时应捕获 `FileNotFoundError`（文件未找到），而不是 `FileExistsError`（文件已存在）。`FileExistsError` 通常在使用 `'x'` 模式创建已存在文件时抛出。

---

#### 错误 3：正则表达式分割模式错误（第 11 行）

```python
# 错误
words = re.split(r"[ \w'-]+", content.lower())

# 修正
words = re.split(r"\W+", content.lower())
```

**解释：** 原正则 `[ \w'-]+` 把单词字符 `\w` 也包含在分隔符中，会把单词本身分割掉。应该用 `\W+`（非单词字符）作为分隔符来正确分割单词。

---

#### 错误 4：打印语句中格式化错误（第 13 行）

```python
# 错误
print(f"In the file {filename}, the word appears {word}")

# 修正
print(f"In the file {filename}, the word '{word}' appears {num_repetitions} times.")
```

**解释：** 原代码打印的是单词本身 `word`，而不是出现次数 `num_repetitions`。应该打印计数结果 `num_repetitions`。

---

#### 错误 5：`encoding='ascii'` 编码设置不当（第 4 行）

```python
# 错误
with open(filename, encoding='ascii') as f:

# 修正
with open(filename, encoding='utf-8') as f:
```

**解释：** `alice.txt`（《爱丽丝梦游仙境》）很可能包含非 ASCII 字符。使用 `encoding='ascii'` 会抛出 `UnicodeDecodeError`。应使用 `encoding='utf-8'` 来支持更广泛的字符集。

> **注：** 5 个错误的判定可能因老师而异。另一种常见的第 5 个错误是：`try...except...else` 中 `else` 块在异常时不执行，但若文件不存在则 `content` 未定义，后续代码会报 `NameError`。建议在 `except` 块中加 `return` 提前退出。

---

### 修正后的完整代码

```python
import re

def count_repetitions(filename, word):
    try:
        with open(filename, encoding='utf-8') as f:
            content = f.read()
    except FileNotFoundError:
        print(f"The file {filename} does not exist.")
        return
    except UnicodeDecodeError:
        print(f"Encoding issue in {filename}")
        return
    words = re.split(r"\W+", content.lower())
    num_repetitions = words.count(word.lower())
    print(f"In the file {filename}, the word '{word}' appears {num_repetitions} times.")

count_repetitions('alice.txt', 'Wonderland')
```
